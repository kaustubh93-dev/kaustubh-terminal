# Kaustubh Terminal Portfolio

A terminal-style portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features
- 50+ interactive terminal commands
- Multiple themes (dark, light, hacker)
- Weather display, Matrix rain, Hollywood hacker mode
- Auto-complete, command history, inline hints
- Shutdown/resume simulation with sound effects

## Commands
Type help to see all available commands.

## Hosting
Deploy as a static site on Vercel, Netlify, or GitHub Pages.
